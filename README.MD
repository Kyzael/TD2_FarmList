# Division 2 — Farm Task List
 
A self-contained HTML tracker for farming the best PvE DPS builds in The Division 2 (2026 meta), including SHD Watch and seasonal leveling builds.
 
**Live site:** `https://Kyzael.github.io/TD2_FarmList`
 
## What's included
 
- farming tasks organized by activity (Countdown, Summit, Raid, Dark Zone, etc.)
- Filters by activity, build, and completion status, plus text search
- Each task expands to show farming steps and direct links to the Division 2 wiki
- Important notes section covering gotchas like raid-locked exotics and the Chatterbox quest chain
- Progress tracker with localStorage persistence (your checkboxes save between visits)

## Builds covered
 
| Build | Role |
|---|---|
| Striker DPS | Primary meta DPS — AR/SMG with St Elmo's Engine |
| Heartbreaker DPS | AR/shotgun with Flatline synergy |
| AR DPS | Coyote's Mask + St Elmo's flexible DPS |
| Pestilence LMG | Negotiator's Dilemma or Coyote's + Pestilence plague |
| SMG Speed Farm | Hunter's Fury + Chatterbox + Memento |
| SHD/XP Farm | Hotshot sniper build for fast SHD Watch leveling |
